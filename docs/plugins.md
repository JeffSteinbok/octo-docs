# Plugins

This page lists the plugins Octo uses today and links to the right docs for each one.

- **Full pages in Octo Docs:** 5
- **Linked to external docs:** 4

## Plugins with full pages here

These plugins have full documentation in Octo Docs.

| | Plugin | Description | Tools |
|---|--------|-------------|:-----:|
|  | [FastMail tools](plugins/fastmail) | Send email and manage calendar events in Fastmail | 7 |
|  | [Home Assistant](plugins/homeassistant) | Control devices, query state, and inspect activity in Home Assistant | 11 |
|  | [Home Assistant – LLM Vision](plugins/llmvision) | Home Assistant LLM Vision integration: analyze camera images with AI, query the vision timeline, and create timeline events. | 4 |
|  | [Package Tracking](plugins/package-tracking) | Track packages from UPS, FedEx, USPS, and Amazon | 5 |
|  | [Stock Quotes](plugins/stock-quotes) | Fetch current stock, ETF, and mutual fund quotes | 2 |
